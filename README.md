# HealthPulse_Hospital_No-Show_Intelligence

Hospital No-Show Prediction System

An end-to-end machine learning system designed to predict patient appointment no-shows, reduce revenue leakage, and optimize healthcare operations — deployed with API, MLflow tracking, and an interactive dashboard.

🚀 Overview

Healthcare providers face significant operational and financial challenges due to patient no-shows.

This project builds a predictive intelligence system that transforms historical appointment data into actionable insights to:

Predict likelihood of patient no-shows
Improve scheduling efficiency
Optimize provider utilization
Reduce wait times and operational inefficiencies
🎯 Business Objectives
Reduce Revenue Leakage
→ Address ~$1.2M annual loss caused by missed appointments
Improve Patient Experience
→ Reduce average wait time from 45 minutes → target 15 minutes
Balance Provider Utilisation
→ Resolve imbalance (95% vs 40% workload distribution)
Enable Predictive Scheduling
→ Shift from reactive to proactive appointment management
⚠️ Current Operational Challenges
22% No-Show Rate (Industry Avg: 12%)
$1.2M Annual Revenue Loss
45-Minute Average Wait Time
Uneven Provider Utilisation
Reactive Scheduling System
🧠 Approach

The solution combines:

Behavioral analytics (patient history)
Operational metrics (provider & clinic load)
Temporal patterns (time, day, scheduling gaps)
Machine learning (Random Forest classifier)
